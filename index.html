Học về CSS

CSS(Cascading style sheets) là ngôn ngữ định hình phong cách cho website.

Các vị trí đặt CSS

  1. Inline(Nội tuyến) là cách viết CSS ngay bên trong thẻ mở của một phần tử html

    2. Với cách đặt này thì CSS chỉ có tác dụng ngay trên thẻ đó.
    Ví dụ: Trong đoạn mã dưới đây 

        
        <h1 style="background-color: brown; color: white">Học về CSS</h1>
        <p style="color: blueviolet">

    Ở đây style: là attribute của html, phần trong nháy "" là ngôn ngữ CSS, background-color và color là các property của CSS, phần sau dấu : là giá trị.

    Nhược điểm của cách đặt này là nếu như trong trường hợp có nhiều đoạn văn bản có cùng property và giá trị giống nhau thì ta phải thêm thủ công vào từng đoạn văn bản, việc này rất tốn thời gian.

  2. Internal(CSS nội bộ) là cách đặt khối lệnh CSS vào trong cặp thẻ <style></style> (bên trong thẻ head) 
        
  <head>
     ...
     <style>
             h1{
                   color: blue;
                   font style: italic;
               }
     </style>
  </head>

    Ưu điểm và nhược điểm của cách đặt này

    • Ưu điểm: code CSS tách biệt với phần tử html, có thể sử dụng toàn trang
    • Nhược điểm: vẫn khó quản lý khi gặp dự án lớn, nhiều mã CSS sẽ làm cho tệp HTML trở nên cồng kềnh.

  3. External(CSS bên ngoài): trong trường hợp muốn áp dụng CSS cho toàn bộ file HTML trong website, tránh copy nhiều lần CSS vào các file đó
    • Tạo file css quản lí riêng trong dự án, ví dụ: style.css
      # style.css 
        
    h2 {
      color: brown;
      font-style: italic;
    }
    • Cần khai báo tệp trong phần tử head của tệp html để link đến tệp css 
        
    <head>
      ...
      <link rel="stylesheet" href="style.css">
    </head>
      Ở đây rel="stylesheet" sẽ thông báo cho trình duyệt tải tệp css đã chỉ định và áp dụng các định dạng kiểu của nó cho cả website.
